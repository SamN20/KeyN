Privacy Policy

Last updated: {{ current_year }}

1. Overview

This Privacy Policy explains how {{ BRAND_PRODUCT }} ("we") processes authentication-related data.

2. Data Collected

  • Account data: username, email, hashed password, profile fields.
  • Security data: passkey public keys, multi-factor tokens, verification timestamps.
  • Operational data: IP, user agent, session + refresh token metadata for security auditing.

3. Purpose

Data is used exclusively for authentication, security monitoring, fraud prevention, and improving reliability.

4. Storage & Security

Passwords are hashed (never stored in plain text). Passkeys store only public key material. Access is restricted.

5. Cookies

Session cookies are scoped to authentication domains and use secure, httpOnly, and SameSite attributes where supported.

6. Sharing

We do not sell personal data. Limited data may be shared with integrated applications strictly for authentication/authorization flows.

7. Retention

Session + token records may be purged after expiration or security review. Account data persists until deletion request.

8. Your Rights

You may request account deletion or email change. Verification is required to process such requests.

9. Changes

We may update this Policy. Material changes will be communicated via login banner or email.

10. Contact

Questions: privacy@bynolo.ca
